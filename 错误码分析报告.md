# 即梦API对外错误码文档

## 错误码列表

| 错误码 | 错误名称 | 错误描述 |
|--------|----------|----------|
| -9999 | API_TEST | API异常错误 |
| -2000 | API_REQUEST_PARAMS_INVALID | 请求参数非法 |
| -2001 | API_REQUEST_FAILED | 请求失败 |
| -2002 | API_TOKEN_EXPIRES | Token已失效 |
| -2003 | API_FILE_URL_INVALID | 远程文件URL非法 |
| -2004 | API_FILE_EXECEEDS_SIZE | 远程文件超出大小 |
| -2005 | API_CHAT_STREAM_PUSHING | 已有对话流正在输出 |
| -2006 | API_CONTENT_FILTERED | 内容由于合规问题已被阻止生成 |
| -2007 | API_IMAGE_GENERATION_FAILED | 图像生成失败 |
| -2008 | API_VIDEO_GENERATION_FAILED | 视频生成失败 |
| -2009 | API_IMAGE_GENERATION_INSUFFICIENT_POINTS | 即梦积分不足 |

## 错误响应格式

所有错误响应均采用统一格式：

```json
{
  "code": -2001,
  "message": "请求失败",
  "data": null,
  "statusCode": 200
}
```

**字段说明**:
- `code`: 错误码，负数表示错误
- `message`: 错误描述信息
- `data`: 附加数据，通常为null
- `statusCode`: HTTP状态码，始终为200

## 常见错误码说明

### -2000 请求参数非法
请求参数不符合规范或缺少必要参数。

### -2001 请求失败
通用请求失败，包括网络错误、服务器错误等。

### -2002 Token已失效
认证Token已过期，需要重新获取。

### -2003 远程文件URL非法
提供的文件URL格式不正确或无法访问。

### -2004 远程文件超出大小
上传的文件大小超过限制。

### -2005 已有对话流正在输出
当前对话正在进行流式输出，请等待完成。

### -2006 内容由于合规问题已被阻止生成
生成的内容违反了内容规范，已被拦截。

### -2007 图像生成失败
图像生成过程中出现错误。

### -2008 视频生成失败
视频生成过程中出现错误。

### -2009 即梦积分不足
账户积分不足，无法完成生成任务。
